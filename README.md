## AtCoder UI cleaner
A userscript to hide certain UI elements from contest page for the reduced amount of noise. 

### Installation
[Greasy Forks link](https://greasyfork.org/en/scripts/572015-atcoder-ui-cleaner)

### Noise
- Score
- Time limit
- Memory limit
- Submission Detail
- Problem Label

### Limitations
- Only works when the site language is set to Japanese
